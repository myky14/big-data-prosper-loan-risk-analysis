# Big Data Prosper Loan Risk Analysis

## Project Overview

This project analyzes and predicts customer credit risk using Apache Hadoop and Apache Spark on the Prosper Loan Dataset.

The project was developed for the Big Data course and demonstrates the complete Big Data workflow:

* Data storage using Hadoop HDFS
* Distributed data processing using Apache Spark
* Advanced data analysis using Spark SQL
* Machine Learning using Spark MLlib
* Business insights and practical recommendations

## Dataset

Dataset: Prosper Loan Dataset

Source:
https://www.kaggle.com/datasets/nurudeenabdulsalaam/prosper-loan-dataset

Dataset Characteristics:

* Records: 113,937+
* Features: 81
* Domain: Financial Services / Credit Risk

## Project Objectives

* Analyze factors affecting loan performance
* Identify customer risk patterns
* Generate business insights using Spark SQL
* Build a machine learning model to predict loan risk indicators
